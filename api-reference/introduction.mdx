## Base URL

```
https://app.contentmaker.space/api/v1
```

## Authentication

All API requests require a Bearer token. See [Authentication](/api-reference/authentication) for details.

## Rate limits

The API is rate-limited to **60 requests per minute** per token. Exceeding this returns a `429 Too Many Requests` response.

## Response format

All responses are JSON. Successful responses return a `data` key; errors return `error` and `message` keys.

```json
{
  "data": { ... }
}
```

```json
{
  "error": "not_found",
  "message": "Article not found"
}
```

## Versioning

The current API version is `v1`. Breaking changes will be released under a new version with a deprecation notice.
