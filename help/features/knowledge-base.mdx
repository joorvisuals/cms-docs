## Overview

The knowledge base lets you create articles scoped to a specific company. Articles are searchable and can be used to document processes, onboarding guides, or deliverables.

## Creating an article

1. Go to **Knowledge Base** in the sidebar
2. Click **New Article**
3. Select the client company
4. Write your content using the rich text editor
5. Click **Publish**

## Searching articles

Use the search bar at the top of the Knowledge Base section. Results are filtered to the selected company by default.

## Article statuses

| Status | Description |
|--------|-------------|
| Draft | Only visible to your team |
| Published | Visible to the client (if sharing is enabled) |
| Archived | Hidden from default views |

## API access

Articles are available via the [API](/api-reference/endpoints/articles) for programmatic creation and retrieval.
